House Price Prediction

Machine Learning Model with Linear Regression

1. Problem description

The goal of this project is to predict property prices using machine learning techniques, specifically linear regression. In the real estate sector, a property's value depends on multiple factors, such as property size, number of rooms, location, building age, and other relevant characteristics. Estimating a property's appropriate price can be a complex process if done manually. By using a predictive model, it's possible to analyze historical property data and find relationships between property characteristics and their sale price. This allows for faster and more objective estimates based on real data. This type of solution is useful for real estate companies, buyers, and sellers, as it facilitates informed decision-making.

2. Explanation of the data set

The dataset used contains information on different real estate properties and their main characteristics. Each row represents a property, and each column represents a variable that can influence its final price. The dataset includes structural variables such as square footage, number of bedrooms and bathrooms, number of floors, garage availability, age, and distance from the city center. It also includes the target variable price, which represents the property's value in Colombian pesos. This data allows the machine learning model to be trained to identify patterns and relationships between property characteristics and their prices, which in turn enables predictions about new properties.

3. Variables used

Dependent variable: price → represents the property's value in Colombian pesos. Independent variables:

  • Area_m² → surface area in square meters
  • Bedrooms → total number of bedrooms
  • Bathrooms → number of bathrooms
  • Floors → number of floors
  • Garage → indicates if the house has garage
  • Age → property age
  • Distance_center_km → distance from city center

These variables allow the model to analyze how each characteristic influences the final property price.

4. Data preparation

Before training the machine learning model, the dataset had to be prepared. First, the file containing the housing information was loaded using the pandas library. Then, the independent variables were separated from the dependent variable (price). After organizing the data, the linear regression model was trained using scikit-learn. The model analyzes the relationship between the input variables and the housing price to find a mathematical function that allows predictions. Data preparation is a fundamental step in machine learning because it ensures the information is correctly structured.

5. Regression graph

The regression graph shows the relationship between one independent variable and the price. In this case, we observe how the area of the house influences the final value. Generally, larger houses tend to have higher prices.

6. Interpretation of results

The linear regression model estimates house prices based on their characteristics. After training with historical data, the model identifies patterns that explain how each variable affects price. For example: larger houses usually cost more, houses closer to the city center tend to have higher value, properties with garage may increase price. The prediction is not exact, but it provides a useful estimate for decision making in real estate.
