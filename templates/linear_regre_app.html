House Price Prediction

Machine Learning Model with Linear Regression

1. Problem description

The goal of this project is to predict property prices using machine learning techniques, specifically linear regression. In the real estate sector, a property's value depends on multiple factors such as size, number of rooms, location, and age. A predictive model allows faster and more objective price estimation based on historical data.

2. Explanation of the data set

The dataset contains information about different properties and their main characteristics. Each row represents a property and each column represents a variable influencing price.

3. Variables used

  • Area_m²
  • Bedrooms
  • Bathrooms
  • Floors
  • Garage
  • Age
  • Distance_center_km

4. Data preparation

Data was loaded using pandas and separated into input variables and target variable. Then the linear regression model was trained using scikit-learn.

5. Regression graph

The graph shows the relationship between area and price.

6. Interpretation of results

The model estimates house prices based on characteristics like size, rooms, and location. Larger houses tend to have higher prices.
