01

What is linear regression?

Linear regression is a statistical and machine learning method used to predict a numerical value from one or more input variables. Its purpose is to identify a linear relationship between variables and represent that relationship with a straight line.

In simple terms, the model tries to answer this question: how does one variable change when another variable changes? For example, if the size of a house increases, does its price also increase? Linear regression helps quantify that trend.

General equation:
y = mx + b
02

Independent and dependent variables

Linear regression works mainly with two kinds of variables:

  • Independent variable (X): the input or factor used to explain or predict another value.
  • Dependent variable (Y): the outcome we want to estimate or predict.

Example

In a real estate scenario:

  • X: house area in square meters
  • Y: property price

As the area changes, the price may also change. Linear regression is used to model and measure this relationship.

03

Training data

Training data is the set of examples used to teach the model. Each row contains known inputs and the corresponding real output. By analyzing those examples, the model learns a pattern that can later be used for prediction.

The better and more representative the data is, the more reliable the predictions will be.

04

Regression line

The regression line is the straight line that best fits the observed data points. It does not need to pass exactly through every point, but it should stay as close as possible to the overall trend.

Once the line is found, it can be used to estimate unknown values. If we know a new value of X, we can use the equation of the line to predict Y.

Key idea: the regression line summarizes the relationship between variables in a single equation.
05

Slope and intercept

In the equation y = mx + b, each element has a meaning:

  • m = slope: shows how much Y changes when X increases by one unit.
  • b = intercept: the value of Y when X is equal to 0.

Example

y = 2x + 10

Here, for every 1-unit increase in x, the value of y increases by 2. When x = 0, the predicted value of y is 10.

06

Application example

One of the most common applications is property price prediction. Imagine a real estate company that has historical information about houses, including area, number of rooms, location, and price.

If a simple linear regression model uses only the area as the independent variable, the model can estimate the price of a new house based on its square meters.

Practical example:
If the model learns that larger houses usually have higher prices, then a 90 m² home can be assigned an estimated price according to the trend observed in the training data.

This model is also useful in business, economics, health analytics, digital marketing, and demand forecasting.

Conclusion

Linear regression is one of the best starting points in data science because it is simple, interpretable, and highly useful. It helps us understand how variables are connected and how historical data can be turned into meaningful predictions for real-world decisions.
