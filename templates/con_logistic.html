Logistic Regression

Understanding the fundamental concepts

1

What is Logistic Regression?

Logistic regression is widely applied in fields such as data analysis, machine learning, and business forecasting to understand relationships between variables. It uses a logistic function to model relationships and provides probabilities instead of exact numeric predictions.

It simplifies complex data patterns and helps organizations make data-driven decisions. Businesses use logistic regression to identify trends, improve marketing campaigns, optimize customer segmentation, and increase operational efficiency.

2

Difference Between Regression and Classification

Regression predicts continuous values such as temperature or price. Classification predicts discrete class labels such as spam or not spam.

  • Regression example: predicting student height.
  • Classification example: predicting whether an email is spam.
  • Regression output → continuous values.
  • Classification output → probabilities of belonging to a class.

Regression can estimate values with infinite possibilities, while classification determines categories based on probability.

3

Binary Classification

Binary classification assigns observations into two groups according to a rule. Examples include:

  • Medical diagnosis: disease or no disease
  • Quality control: product accepted or rejected
  • Search engines: relevant or irrelevant results

False positives and false negatives are important considerations depending on the problem context.

4

Target Variable / Class Label (y)

The target variable is the value the model aims to predict. It represents the known answer used during training.

Binary values:
1 → event occurs (success)
0 → event does not occur (failure)

Example: Smoker → Yes (1) or No (0)

5

Features / Predictors (X)

Features are measurable variables used as input for prediction. Mathematically:

$$ X = [x_1, x_2, x_3, ..., x_n] $$

Each feature has a weight that represents its importance.

  • Positive weight → increases probability
  • Negative weight → decreases probability

The bias term allows the decision boundary to shift.

6

Probability Estimation (ŷ)

Logistic regression estimates the probability that a data point belongs to Class 1.

$$ z = w_1x_1 + w_2x_2 + ... + w_nx_n + b $$
$$ \sigma(z) = \frac{1}{1 + e^{-z}} $$

The output value ranges between 0 and 1. Example: 0.92 means a 92% probability of belonging to the positive class.

7

Classification Threshold

The threshold converts probability into a class label.

P ≥ 0.5 → Class 1
P < 0.5 → Class 0

Threshold selection depends on context:

  • Medical diagnosis → lower threshold to avoid missing disease
  • Legal decisions → higher threshold for certainty
8

Decision Boundary

The decision boundary separates classes based on probability.

In simple logistic regression, the boundary is linear. Points on one side belong to Class 1, while points on the other belong to Class 0.

9

Real-World Example: Customer Churn

A telecommunications company predicts whether a customer will cancel their subscription.

Features:
x₁ → contract duration
x₂ → monthly cost
x₃ → number of service calls

If the model predicts probability = 0.78 and the threshold = 0.6, the customer is classified as high risk.

The company can offer promotions to retain the customer.
